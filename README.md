## 🚀 Featured Project

### ⚡ Power Supply Design (Dual Rail 5V / 3.3V)

#### 📖 Description

Designed a dual-output regulated power supply capable of delivering stable 5V and 3.3V outputs. The system maintains voltage stability under varying load conditions and minimizes ripple through proper filtering and regulation.

---

#### ⚙️ Key Features

- Dual voltage rails (5V and 3.3V)  
- Linear voltage regulation  
- Input filtering and decoupling  
- Stable performance under dynamic loads  

---

#### 🧪 Simulation & Analysis

Simulated using SPICE to evaluate:

- Load regulation  
- Voltage ripple  
- Transient response during step load changes  

---

#### 🧪 Test Scenario

Simulated load conditions:

- No load (0mA)  
- Medium load (250mA)  
- Full load (500mA)  

Step load transitions were applied to evaluate voltage stability and recovery performance.

---

#### 📊 Results

- **5V Rail Output:** 4.95V – 5.05V  
- **3.3V Rail Output:** 3.25V – 3.35V  
- **Ripple Voltage:** 20mV – 50mV  
- **Transient Response:** < 2ms recovery time  
- **Load Stability:** Maintained across 0mA – 500mA  

---
